# ...And Gretel — Advanced Flipbook

Version rebuilt with **PageFlip.js** for smoother page turns, better spread handling, and improved responsiveness.

## How to run

1. Open this folder in **Visual Studio Code**.
2. Install/enable **Live Server**.
3. Open `index.html` with Live Server.

## Important

This version uses **PageFlip.js from a CDN**.
So the first load needs an internet connection.

If you want, you can later replace the CDN script with a local copy of the library.
